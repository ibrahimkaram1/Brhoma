# 💼 Hime Profile Portfolio

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript**.  
This portfolio showcases personal information, skills, projects, and contact details in a clean and minimal design.

## 🌐 Live Demo
🔗 https://hime-profile.netlify.app/

---

## 📌 Features

- Responsive design (works on desktop, tablet, and mobile)
- Smooth scrolling navigation
- Animated sections
- Skills and projects showcase
- Contact section
- Clean and modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Netlify (for deployment)
